Welcome to Park It ExpressJS server!

Here is a small documentation of all the routes of our backend server.

Category Type Route Scope Function
Users POST /api/users/register null Register a new user
POST /api/users/login null Login a existing user
GET /api/users/profile user Get the user profile
PUT /api/users/profile user Update the user profile
POST /api/users/admin admin Register a new admin
GET /api/users/all admin List all the registered users
GET /api/users/:id admin Get a user by ID
DELETE /api/users/:id admin Delete a user by ID
Areas GET /api/areas user Get all the areas
GET /api/areas/:id user Get all the slots by the area ID
POST /api/areas admin Create a new area
PUT /api/areas/:id admin Update a existing area
PUT /api/areas/:id/slot/:slotName admin Reverse the slot data
DELETE /api/areas/:id admin Delete area by ID
Bookings POST /api/booking/:id/:slotName user Create a new booking
GET /api/booking user Get all the bookings of a user
GET /api/booking/all admin Get all the bookings
GET /api/booking/:id admin Get the booking by ID
PUT /api/booking/:id admin Update a existing booking
